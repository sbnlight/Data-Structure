# Data-Structure

COMP2119 Introduction to Data Structures and Algorithms — The University of Hong Kong.

C++ implementations for course assignments. Each `.cpp` file is a standalone problem solver that reads input from stdin.

## Contents

| Folder | Topic |
|--------|-------|
| `Graph/` | Graph algorithms — BFS, Dijkstra |
| `Tree/`  | Tree data structures — BST, AVL |

See each subfolder's README for problem descriptions.

## Build

```bash
g++ -std=c++17 -O2 path/to/file.cpp -o solution
./solution < input.txt
```
